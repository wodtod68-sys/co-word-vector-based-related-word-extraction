# 공기어 벡터 기반 관련어 추출 (Co-word Vector based Related Word Extraction)

대상어-공기어 쌍의 t-점수로부터 단어 벡터(공기어 벡터)를 구축하고,
코사인 유사도를 이용해 사용자 질의어와 가장 유사한 관련어를 추출하는 도구.


--------------------------------------------------------------------------------
 구성 파일
--------------------------------------------------------------------------------

  coword_vector_indexer.py    공기어 벡터(단어 벡터)를 생성하여 pickle로 저장
  coword_vector_similarity.py  질의어와 유사한 상위 30개 관련어를 출력


--------------------------------------------------------------------------------
 1. coword_vector_indexer.py
--------------------------------------------------------------------------------

대상어와 공기어 쌍의 t-점수가 저장된 입력 파일로부터
단어 벡터(공기어 벡터)를 생성한다.
(참고: 코퍼스 분석 로드맵.pptx p.28~29)

  [실행 방법]
    $ ./coword_vector_indexer.py 입력파일 출력파일
    예) $ ./coword_vector_indexer.py all.tscore all.pickle

  [입력 파일 형식]
    탭(\t)으로 구분된 3개 필드로 구성된 텍스트 파일.
    필드 수가 3이 아닌 줄은 무시한다.

        대상어 <TAB> 공기어 <TAB> t-점수

  [출력 파일 형식]
    dictionary의 dictionary 구조를 갖는 pickle 파일.

        { 대상어 : { 공기어 : t-점수(float), ... }, ... }


--------------------------------------------------------------------------------
 2. coword_vector_similarity.py
--------------------------------------------------------------------------------

사용자 질의(query)와 가장 유사한 30개의 단어(관련어)와 유사도를 보여준다.
유사도는 두 공기어 벡터 간 코사인 유사도(cosine similarity)로 계산한다.
(참고: 코퍼스 분석 로드맵.pptx p.30~32)

  [실행 방법]
    $ ./coword_vector_similarity.py 입력파일
    예) $ ./coword_vector_similarity.py all.pickle

  [입력 파일]
    coword_vector_indexer.py가 생성한,
    단어 벡터(공기어 벡터)가 저장된 pickle 파일.

  [사용 방법]
    실행 후 프롬프트에 검색할 단어를 입력한다.
    질의어와 유사한 상위 30개 관련어가 유사도 내림차순으로 출력된다.
    종료하려면 ^D (Ctrl+D)를 입력한다.

        검색할 단어를 입력하세요(type "^D" to exit):
        <질의어 입력>

        관련어 <TAB> 유사도(소수점 3자리)
        ...

  [제약 사항]
    - 관련어 후보는 대상어의 공기어들과,
      그 공기어들의 공기어들로 한정한다.
    - 코사인 유사도가 0.001보다 큰 경우만 저장한다.
    - 관련어 후보가 대상어와 같으면 출력하지 않는다.
    - 관련어 후보가 대상어에 포함되면 출력하지 않는다.
        예) 개인정보/정보, 교육부장관/장관, 인사청문회/청문회
    - 유사도 내림차순으로 상위 30개 관련어를 출력한다.


--------------------------------------------------------------------------------
 실행 순서 (예시)
--------------------------------------------------------------------------------

  # 1단계: t-점수 파일로부터 공기어 벡터 pickle 생성
  $ ./coword_vector_indexer.py all.tscore all.pickle

  # 2단계: 생성된 pickle을 이용해 관련어 검색
  $ ./coword_vector_similarity.py all.pickle

      $ chmod +x coword_vector_indexer.py coword_vector_similarity.py
  또는 python3으로 직접 실행:
      $ python3 coword_vector_indexer.py all.tscore all.pickle
